About the person behind the tabs

如果要认真介绍我，我不想只交出一串标签。

我希望这个网站既能成为“简历里的链接”，也能像一个慢慢长出来的私人空间。 所以它不会只放一份静态简历，而是把我已经在不同平台上认真做的内容重新整理到一起： GitHub 上的项目、小红书上的不同身份、公众号里的长文，以及那些会慢慢长成个人风格的兴趣和判断。

在职业面上，我会把这里当成项目与思考的统一入口：一部分内容来自 GitHub， 展示我真正动手做过的项目；另一部分来自我对算法、产品和长期问题的写作。 我很喜欢“算法与产品并行，计算与思考同在”这句话，因为它确实比一串技能标签更接近我的状态。

在生活面上，我也不想把自己剪成只有“职业身份”的一块。我的美食账号、日常观察、 对城市和味觉的记录，和我做项目、写长文一样，都是同一个人的表达。两个小红书账号彼此相连， 一个更靠近味觉和生活，一个更靠近算法、产品与长思文章；这个网站想做的，就是把它们重新放回同一个人身上。
